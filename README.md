# Replication Materials for "An Epidemiological-Modeling and Time-Series Forecasting Approach for Hospital Patient Allocation during Pandemics" (Abazari et al., 2025)

This repository provides the replication code and supporting materials for the publication:

**Abazari et al. (2025), "An Epidemiological-Modeling and Time-Series Forecasting Approach for Hospital Patient Allocation during Pandemics"**

## Repository Structure

- **SIR/**
  - A rendered Markdown file is included as a README to illustrate the model outputs.

- **Time Series Forecasting/**
  - A rendered Markdown README file is provided to demonstrate the results.

## Purpose

The goal of this repository is to ensure transparency and reproducibility of the analyses presented in the publication. All scripts and Markdown reports can be executed using the provided data and instructions.
